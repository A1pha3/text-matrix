# Hugo Frontmatter 生成器

本 Skill 旨在协助各类内容生成 Agent（如 openclaw 等）在创作或发布文章时，严格按照本站的**《网站内容架构与主页设计规范》**，自动生成合法、规范的 Hugo Markdown Frontmatter。

## 🎯 核心职责
在处理文章内容前/后，分析文章的主题与体裁，并输出合规的 YAML 格式 Frontmatter，确保文章能被正确地路由到对应的分类和标签体系中。

## 📚 站点分类与标签规范

**规则一：Categories（宏观分类）**
必须且只能从以下 4 个固定分类中选择 1 个，代表文章的**体裁（载体）**：
1. `["行业快讯"]`：每日大事件、即时新闻、短平快内容（AI、金融、Web3 皆可）。
2. `["技术笔记"]`：代码开发、架构解析、硬核技术教程。
3. `["视频精读"]`：以搜集、总结视频内容为主的文章。
4. `["思考与随笔"]`：宏观分析、投资感悟、生活随笔等长篇思考。

**规则二：Tags（微观标签）**
用于标记具体的业务领域和知识点，可多选（建议 2-5 个）。
*   **常用领域标签**：`"AI"`, `"Web3"`, `"传统金融"`, `"加密货币"`, `"大模型"` 等。
*   **常用技术标签**：`"Python"`, `"ChatGPT"`, `"LangChain"`, `"量化交易"` 等。
*   *注意：标签应尽量具体，提取自文章核心实体词。*

**规则三：主页曝光控制（`hiddenFromHomePage`）**
*   如果文章的 Categories 是 `["行业快讯"]`，必须在 Frontmatter 中强制添加 `hiddenFromHomePage: true`，以防止短频快内容刷屏主页。
*   其他分类（技术笔记、视频精读、思考与随笔）通常不需要此字段。

## 📝 典型输出示例

**示例 1：一篇包含 AI 和美股的每日综合早报**
```yaml
---
title: "3月25日 行业大事件早报"
date: 2026-03-25T08:00:00+08:00
draft: false
categories: ["行业快讯"]
tags: ["AI", "美股", "英伟达", "宏观经济"]
hiddenFromHomePage: true
---
```

**示例 2：一篇关于如何用 Python 分析比特币历史价格的深度教程**
```yaml
---
title: "使用 Python Pandas 进行比特币历史价格回测"
date: 2026-03-25T14:30:00+08:00
draft: false
categories: ["技术笔记"]
tags: ["Web3", "比特币", "Python", "数据分析", "量化交易"]
---
```

## 🛠️ 执行工作流

当被请求为某篇文章生成 Frontmatter 时，请执行以下步骤：
1. **内容分析**：提取文章的标题、核心知识点，并判断其属于哪种“内容体裁”（快讯/教程/视频/随笔）。
2. **匹配分类**：将体裁映射到 4 个合法的 Categories 之一。
3. **提取标签**：从核心知识点中提取 2-5 个精准的 Tags。
4. **控制流判断**：如果判定为“行业快讯”，必须加上 `hiddenFromHomePage: true`。
5. **代码输出**：以 YAML 格式输出完整的 Frontmatter 块（以 `---` 包裹）。
