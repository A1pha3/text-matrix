# 关于 Text Matrix

Text Matrix 是一个面向中文开发者的 AI Agent、AI 开发工作流与开源工具知识库。

这个站点关注的不是泛泛的 AI 资讯，而是那些真正值得理解、测试、落地和长期追踪的内容：AI Agent、开发工具链、CLI 工作流、开源项目、工程实践，以及少量与这些主题强相关的行业变化。

这里会持续更新几类内容：

- AI Agent 与 AI 开发工作流的系统化讲解
- 开源项目、开发工具与工程实践的深度解读
- 值得跟进的行业快讯与视频精读
- 少量围绕独立开发、内容系统和长期积累的思考

如果你是开发者、独立开发者、AI 工具重度使用者，或者正在建立自己的技术工作流，这里的内容会更适合你。

Text Matrix 想做的不是短期热点搬运，而是把高价值的信息整理成可检索、可复用、可持续更新的中文内容资产。
