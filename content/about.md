# 关于 Text Matrix

Text Matrix 是一个面向中文开发者的站点，专门跟踪 AI Agent、AI 开发工作流和值得长期使用的开源工具。

## 主要内容

内容集中在五类：

- AI Agent 的拆解，从概念边界到生产化细节
- 开发工具链、CLI 工作流、开源项目的源码级分析，重点看怎么搭起来、边界在哪
- 行业里值得跟进的快讯和视频
- 关于独立开发、内容系统、长期积累的零散想法

## 目标读者

目标读者是：

- 用 Claude Code、Codex、Cline 这类 Coding Agent 写代码的工程师
- 想搭一套自己的 AI 工作流、不想被某个 SaaS 绑死的独立开发者
- 长期跟踪 AI 工具变化、想自己判断哪些值得长期跟的读者

## 想做的事

把分散在 GitHub、YouTube、X 上的资料，整理成可以反复查、可以串进工作流、一段时间之后还值得再翻的中文版本。
