来源: 36kr
原文: [原文](https://www.36kr.com/p/3868515171718148)
摘要: 黄仁勋、Peter（龙虾之父）、Boris Cherny（Claude Code 之父）、吴恩达等一众大佬在最近一周密集推广 loop 范式。黄仁勋的原话是「现在根本没有人写 Prompt 了，新时代的核心工作是编写和管理 loop」。loop 的核心是把人从流程中抽离：你定义一个目标，AI 自己跑、自己验收、不合格就带着报错再来一轮，直到通过或撞上预算上限。Claude Code 已围绕 loop 做了 /loop（定时）、/goal（目标驱动）、/schedule（云端定时）三件套，/goal 内置写代码是大模型、验收是独立小模型 Haiku 的解耦设计，避免 Agent 自己给自己打高分。OpenAI Codex 也在同一方向演进，loop 越过了「又一个新概念」的阶段，开始沉淀为产品架构。
