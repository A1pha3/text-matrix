# dayanch96/YTLite：iOS 版 YouTube 增强工具，支持下载视频/音频、自定义界面

## 🎯 概述

**YTLite**（原名 YouTube Plus）是 iOS 上的 YouTube 增强工具，提供超过百种可自定义选项，让用户无需越狱即可增强 YouTube 体验。

> **GitHub**: [dayanch96/YTLite](https://github.com/dayanch96/YTLite)  
> **Stars**: 4,893 ⭐  
> **最新版本**: 5.2  
> **支持 YouTube**: 21.15.5

---

## ⚡ 主要功能

| 功能 | 说明 |
|------|------|
| **多媒体下载** | 视频、音频（可选择音轨）、缩略图、帖子、个人资料图片 |
| **界面定制** | 移除 Feed 元素、重新排序标签、启用 OLED 模式、Shorts 仅限模式 |
| **播放器设置** | 手势、默认质量、首选音轨 |
| **内置 SponsorBlock** | 自动跳过赞助内容 |
| **画中画** | YouPiP tweak 支持画中画功能 |

---

## 🛠️ 使用方式

### GitHub Actions 自动构建

1. Fork 本仓库
2. 在仓库 Settings → Actions 中启用 Read and Write 权限
3. 进入 Actions → Create YouTube Plus app
4. 运行 workflow 并配置 tweaks
5. 上传解密的 .ipa 文件
6. 等待构建完成，从 releases 下载应用

### 系统要求

- iOS 设备（已解密）
- 非国区 Apple ID（下载第三方应用）
- 证书签名工具（如 Sideloadly）
