§ 概念解析：什么是人格蒸馏

. 核心定义

**人格蒸馏（Persona Distill）是什么？**

> 从对话、作品、资料或数字痕迹中提炼表达风格、决策框架与交互方式。

**重要边界**：人格蒸馏**不默认等同于对真实个体的完整还原**，而是一种方法论视角的提取与重现。

```
┌─────────────────────────────────────────────────────────────┐
│                   人格蒸馏 vs 数字克隆                      │
├─────────────────────────────────────────────────────────────┤
│  人格蒸馏 ✅                                               │
│  ├── 表达风格：词汇选择、句式习惯、语气特征               │
│  ├── 决策框架：判断标准、优先级、启发式                   │
│  ├── 交互方式：回应模式、情绪边界、话题偏好               │
│  └── 适用场景：技能构建、角色扮演、记忆重建               │
│                                                              │
│  数字克隆 ❌（非本项目目标）                               │
│  ├── 完整人格复制                                         │
│  ├── 隐私侵犯                                             │
│  └── 未经授权的人格使用                                   │
└─────────────────────────────────────────────────────────────┘
```

. 为什么需要人格蒸馏

| 需求 | 场景 | Skills 价值 |
|------|------|-----------|
| **职场沟通** | 向上管理、跨部门协作 | 理解老板/同事的决策风格 |
| **亲密关系** | 情感整理、关系回顾 | 数字化保存共同记忆 |

| **纪念意义** | 数字遗产、亲情保存 | 留住与逝者的对话可能 |

§ 自我蒸馏与元工具

. 概述

自我蒸馏是从自身数据中提炼结构化自我画像，用于：
- 记忆整理与个人知识管理
- 自我镜像与反思
- 与他人协作时的风格说明

. 核心 Skills 详解

.. 自己.skill

**仓库**：notdog1998/yourself-skill

**功能**：将个人对话与记录整理为自我蒸馏助手

**使用场景**：
```markdown
当你需要自我反思时
用户：我最近在项目中和同事有分歧
助手（自己.skill）：根据你过去的沟通风格，
你倾向于先理性分析再表达情绪。
这次分歧可能是因为对方打破了你的"预期边界"...
```

**实现要点**：
- 整合聊天记录、会议纪要、决策日志
- 提炼表达模式（理性/感性、主动/被动）
- 构建个人方法论框架

§ 技术实现与 Agent Skills 标准

. Agent Skills 开放标准

所有 Skills 遵循 Agent Skills 规范：

```
skill-name/
├── SKILL.md           # 核心定义（必需）
├── scripts/           # 可执行脚本
│   └── index.html     # JavaScript 技能入口
├── references/        # 参考文档
└── assets/           # 静态资源
```

. SKILL.md 结构

```markdown
---
name: persona-distill
description: 从数字痕迹中提炼人格框架
compatibiity:
  - openclaw

触发条件
当用户请求[具体场景]时触发本技能

核心指令
[详细的技能执行流程]

渐进式披露
[进阶用法，按需加载]
```
