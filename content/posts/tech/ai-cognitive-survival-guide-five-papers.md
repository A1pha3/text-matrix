## 前言

2026年了，ChatGPT、Kimi、Claude这些东西已经到处都是。但有个问题一直没被认真讨论过：我们真的会用法吗？

很多人的用法其实用反了——不是用AI增强自己，而是在被AI替代思考。

MIT Media Lab、微软研究院、卡内基梅隆大学这几个地方做了研究，数据挺说明问题的。

简单说就是一句话：**AI是磨刀石，不是拐杖。**

磨刀石让刀更锋利，用完刀还是你的。拐杖替代腿，用久了腿就废了。

---

## 一、MIT脑电实验

Kosmyna等人在MIT Media Lab做的实验。

他们找了两组人：一组习惯让ChatGPT代写文章，另一组从没用过AI写作。所有人写文章时都用脑电图（EEG）监测大脑活动。

结果很清楚。习惯让AI代写的那组，大脑额叶和颞顶叶区域的活动比对照组低很多。

额叶负责注意力、决策、解决问题。颞顶叶负责工作记忆、语言处理。

换句话说，"省心"的人大脑的思考区域越来越不活跃。

研究者把这种现象叫**认知债务**。你以为省了时间，其实是在透支大脑。像信用卡一样——现在享受，以后还。

还有个后续发现更扎心。那些习惯用AI的被试被要求脱离AI独立写作时，他们的大脑神经连接明显比对照组弱。而且这种弱不是短期能恢复的。

短期效率换长期认知能力，这笔账不划算。

## 二、微软×卡内基梅隆调查

Lee等研究者在微软和卡内基梅隆，调查了319名知识工作者，涵盖多个行业和不同职级。

有个反直觉的发现：

对AI越信任的人，批判性思维投入越少。对自己能力越有信心的人，面对AI时反而思考得更深。

逻辑链是这样的：

```
高信任AI → 低批判性思维投入 → 思考变浅
高自信 → 面对AI时思考更深
```

调查里有个细节挺有意思——对外包式使用AI不以为然的人往往是职位更高、经验更丰富的人。新人反而更容易全盘接受AI输出。

可能因为老手已经有自己的判断体系，AI的答案会先过一遍脑子。新手还没形成这种习惯，AI说什么就信什么。

把思考责任外包给机器，短期内没问题。但独立思考的肌肉长期不用，就会萎缩。

## 三、认知摩擦

2026年ScienceDirect上有篇论文，提了个实用框架，两条：

**第一条：保留认知摩擦。** 思考本来就是费力的。费力的过程本身就是思考的一部分。AI把思考过程抹平了，脑子就不用动了。脑子不动，能力就不长。这跟健身一样——机器帮你做完所有运动，肌肉不会变强。

**第二条：把AI定位为"临时思考伙伴"，不是答案来源。** 不是"有问题找AI"，而是"有答案找AI吵架"。

举个例子。传统用法是：遇到问题 → 问AI怎么做 → 得到答案 → 执行

正确用法是：遇到问题 → 先自己想一个答案 → 让AI来反驳我 → 重新审视 → 得到更好的判断

现在很多人在用的是第一种。上来的第一句话就是"这个问题应该怎么做"——这不是在用AI，这是在交出思考权。

---

## 四、辩论让人判断更准

Khan等人（2024）的研究也发现了类似规律。

他们让AI扮演"辩论对手"，挑战研究参与者的判断。当参与者的观点被AI挑战时，最终得出的结论更准确。

道理很简单。观点没人质疑时，你不会去想它有没有漏洞。有人反驳时，你被迫重新思考：这个反驳有道理吗？我的判断要不要改？

质疑、反驳、重新审视——这就是批判性思维的核心。

所以AI不该只给你答案。AI更应该做的是：挑战你的假设，指出你的漏洞，逼你深入想一个问题。

## 五、结构化反思

微软研究院2026年的实验更直接。

参与者分成两组。一组直接用AI输出，另一组在使用AI前先做结构化反思。

结构化反思问四个问题：

1. 我的判断是什么？
2. 我为什么这样判断？
3. AI的输出可能有什么问题？
4. 我的判断需要修正吗？

就这么四步，结果差距很大。做了反思的人推理质量明显更高，对AI输出的批判性更强，过度依赖AI的程度更低。

而且这些人自己感觉也更好——他们对自己的判断更有信心，这种信心是合理的，不是盲目的。

## 六、判断标准

怎么判断你是在用AI还是被AI用？

|  | AI替你做 | 你做，AI挑刺 |
|---|---|---|
| 思考过程 | AI完成 | 你完成 |
| AI角色 | 答案机器 | 辩论对手 |
| 长期结果 | 越来越懒 | 越来越锐利 |
| | 认知债务 | 认知增值 |

有个更简单的判断：看你使用AI时的第一句话。

上来问"这个问题应该怎么做"——把AI当答案来源。

上来先说"我觉得应该这样做，AI你给我挑挑毛病"——把AI当思考伙伴。

## 七、实操方法

**第一步，先给答案。** 问题再简单，也先逼自己给出一个答案。答案可能错，但比没有强。没有答案时问AI，只是在搬运，不是在思考。

**第二步，让AI来挑刺。** 有自己的答案后，对AI说："我觉得应该这样做。我的思路可能有什么漏洞？"而不是："这个问题应该怎么做？"

**第三步，认真对待AI的反驳。** AI的反驳不一定对，但值得想：AI说的有道理吗？原始判断要不要改？改了之后新判断是什么？

**第四步，迭代不是采纳。** 最终答案还是你的，AI只是帮你打磨。别变成"AI说什么我就信什么"。

## 八、为什么这重要

AI越来越强，但AI输出质量参差不齐。更麻烦的是，AI有种天然倾向——顺着你说，哪怕你的判断是错的。

这不是AI坏掉了，而是它的设计逻辑。AI被训练成要"有用"，而顺着你说通常被认为更有用。

问题在于，没有独立判断能力的话，你分不清AI是在帮你思考还是在讨好你。

当人类知识内化为模型能力，人类该做什么？

答案是：做那个能批判AI、判断AI、使用AI的人，而不是被AI替代判断能力的人。

每天问自己三个问题：

- 今天是在用AI，还是在让AI用我？
- 是在思考，还是在搬运答案？
- 是在变强，还是在变懒？
