## 一、这个仓库为什么会上 GitHub Trending

2026-06-07 这一天，GitHub Trending 上出现了一个**非典型**技术项目：`TapXWorld/ChinaTextbook`。和同榜的 pg_durable、goose、llama.cpp 都不一样——它没有代码、没有算法、没有 AI，**只有一堆 PDF**。

但它依然被推上来，因为 README 第一段话戳中了一个长期被忽视的痛点：

> 虽然国内教育网站已提供免费资源，但大多数普通人获取信息的途径依然受限。有些人利用这一点，在某站上销售这些带有私人水印的资源。为了应对这种情况，我计划将这些资源集中并开源，以促进义务教育的普及和消除地区间的教育贫困。

一句话概括：**国内教材本来是免费提供的，但有人把它加上水印、剪掉页码、当作付费商品卖。ChinaTextbook 就是这个现象的「开源反制」。**

## 三、版权与法律：能不能用、合不合法

这是最常被问的问题，需要分两层看：

### 3.1 教材本身

人民教育出版社（人教版）的中小学教材**是国家组织编写、面向义务教育阶段免费或低价提供的**。教育部、国家发改委等部委曾多次要求「义务教育阶段免费提供教材」，人教社的官网也提供 PDF 下载。

大学教材（如同济版高数）则属于**出版社作品**，仍然受版权保护。但在国内：

- 高校师生用于**教学 / 科研**的合理使用
- 出版社未公开声明禁止下载或仅授权付费渠道

### 3.2 GitHub 仓库的立场

README 明确写「项目由来」：是**为了对抗那些把免费资源加水印再卖的二手商家**，并希望**海外华人孩子能继续了解国内教育**。

这不是洗白也不是擦边——它做的是「把官方已经免费提供的内容集中化、可访问化、可验证化」，避免家长误买盗版。

**实际使用建议**：

- 中小学阶段：放放心心用，原本就是国家免费提供
- 大学阶段：建议先看出版社官方是否提供下载；若无，再使用仓库版本（**仅用于个人学习**）
- 商业用途 / 二次售卖：绝对不要，违反版权

## 四、海外华人场景：仓库存在的「第二个原因」

README 还写了一句被很多转载忽略的话：

> 还有一个最重要的原因是，希望海外华人能够让自己的孩子继续了解国内教育。

这其实是一个被严重低估的群体——

- 2024 年海外华人华侨总数估计 6000 万+
- 大量家庭希望孩子保留中文 + 国内课程体系
- 但中文教材在海外获取渠道极不稳定：
  - 国内邮寄：运费贵、周期长、丢失风险
  - 当当 / 京东海外购：经常被砍单
  - 中文书店：价格高、版本滞后
  - 盗版 PDF 站：带水印、被二次售卖

ChinaTextbook 等于给了海外华人**「一个免费、稳定、可验证的中文教材源」**。这也是它上 trending 的另一个原因——海外开发者看到后主动 star、转发，形成传播。

## 五、技术细节：为什么要在 GitHub 上做这件事

### 5.1 GitHub 的天然优势

- **HTTPS 直链**：README 里每个 PDF 都是 GitHub 的 `blob/master/...` 链接，任何人点开就能下载
- **git pull 增量同步**：上游更新后，下游一份 `git pull` 就能拿到最新
- **版本可追溯**：每次 commit 都可以看到新增 / 替换的 PDF
- **issue / PR 协作**：缺哪个版本直接在 issue 里说
- **git LFS / 普通仓库二选一**：对于几十 MB 的 PDF，**普通 git 仓库 + 35MB 分片**已经能跑（README 明确解释过 GitHub 100MB 上限和 50MB 警告的应对）

### 5.2 大文件分片方案

README 里专门有一段回答「如何合并被拆分的文件」：

> 由于 GitHub 对单个文件的上传有最大限制，超过 100MB 的文件会被拒绝上传，超过 50MB 的文件上传时会收到警告。因此，文件大小超过 50MB 的文件会被拆分成每个 35MB 的多个文件。

这是教科书级别的「在平台规则下做公益」——**绕开限制的同时不违反规则**。

## 七、怎么同步到本地

```bash
# 完整克隆（含所有 PDF，体积较大）
git clone https://github.com/TapXWorld/ChinaTextbook.git

# 只克隆最新一次（节省空间，浅克隆）
git clone --depth 1 https://github.com/TapXWorld/ChinaTextbook.git

# 拉取增量更新
cd ChinaTextbook
git pull

# 拉取大文件版本（如果作者启用了 LFS，需先安装 git-lfs）
git lfs install
git lfs pull
```

**注意**：完整仓库体积可能在数 GB 级别，建议：

- 普通家庭：`--depth 1`
- 教师 / 公益组织：完整克隆 + 本地 CDN
- 做训练集：按目录 sparse checkout，只拉需要的子目录

## 八、和其他「免费教材」项目的关系

- **人教社官网（pep.com.cn）**：源头，**最权威**，但分散在不同栏目
- **国家中小学智慧教育平台（basic.smartedu.cn）**：教育部官方，**含视频课**
- **中国大学 MOOC（icourse163.org）**：大学课程资源
- **ChinaTextbook**：GitHub 上一站式目录，**适合做镜像、做训练集、做海外分发**

它们之间是**互补关系**。

## 九、局限与提醒

- ❌ **不是所有教材都覆盖**：主要人教版，其他版本（苏教版、北师大版）覆盖较少
- ❌ **大学阶段版权更敏感**：商业使用请走出版社授权
- ❌ **PDF 是单层，不是结构化数据**：要做训练集需要自己做 OCR / 段落切分
- ❌ **更新有滞后**：每学期新版上线后，仓库可能延迟几天

## 十、常见问题（FAQ）

### Q1: 下载和使用这些 PDF 合法吗？

中小学教材（人教版）是国家免费提供的公共资源，用于个人学习完全合法。大学教材（如同济高数）版权归出版社，建议仅用于个人学习，不要商业使用。

### Q2: PDF 缺页或模糊怎么办？

在仓库提 issue，社区会更新。也可以去人教社官网（pep.com.cn）或国家中小学智慧教育平台（basic.smartedu.cn）下载官方版本。

### Q3: 海外用户怎么下载最快？

用 `git clone --depth 1` 浅克隆，只拉最新版本。如果 GitHub 访问慢，可以用镜像站或等国内 Gitee 同步（如果有的话）。

### Q4: 能打印这些 PDF 给孩子用吗？

能。很多家长这么做。建议用双面打印省钱，也环保。

### Q5: 怎么参与维护这个仓库？

可以提 PR 补充缺失的教材版本，或修复已有的 PDF（比如页码错误、水印残留）。仓库是公开社区维护的，欢迎贡献。
