## 什么是 AI Engineering from Scratch

目前仅有18%的AI工具用户感觉自己能专业地使用它们——这个课程要填补的就是这个巨大缺口。

**核心数据：**
- 🎓 435节课程
- 📅 约320小时内容
- 🧮 20个递进阶段
- 💻 Python、TypeScript、Rust、Julia 四语言
- 🆓 全部开源，MIT许可

课程从底层数学开始，一路搭到顶层Agent系统：

```
Phase 0 — 环境配置

Phase 2 — 机器学习基础
Phase 3 — 深度学习核心
Phase 4 — 计算机视觉
Phase 5 — 自然语言处理
Phase 6 — 语音与音频
Phase 7 — Transformers
Phase 8 — 生成式AI
Phase 9 — 强化学习
Phase 10 — 从零构建LLM
Phase 11 — LLM工程
Phase 12 — 多模态
Phase 13 — 工具与协议（MCP等）
Phase 14 — Agent工程
Phase 15 — 自主系统
Phase 16 — 多Agent与Swarm
Phase 17 — 基础设施与生产级部署
Phase 18 — 伦理与对齐
Phase 19 — 毕业项目
```

## 每节课的结构

每节课都有固定的结构：

```
phases/<NN>-<phase-name>/<NN>-<lesson-name>/
├── code/      可运行代码实现
├── docs/en.md 课程正文
└── outputs/   产出的 Prompt、Skill、Agent或MCP Server
```

每节课遵循六步循环：**座右铭 → 问题 → 概念 → 从零实现 → 生产级使用 → 反思**，先徒手写算法再对比框架实现，真正理解底层原理。

## 特色亮点

**Build It / Use It 双轨制**：先从原始数学实现（不依赖任何框架），再对比生产级库（PyTorch等），确保理解框架在做什么。

**产出可复用**：每节课产出一个可上线的 artifacts——可以是 prompt、agent、MCP server 或 skill。

**完全自包含**：在自己的笔记本上就能跑完所有内容。
