# RuView：用 WiFi 信号实现无摄像头隐私监控

**Stars: 59,948** | **今日: +700** | **Forks: 7,812** | **Language: Rust**

GitHub: [ruvnet/RuView](https://github.com/ruvnet/RuView)

## 一句话评价

RuView 将任何普通 WiFi 设备（路由器 + 任意设备）变成实时空间感知系统——无需摄像头，用 WiFi 信号的多径效应实现呼吸检测、人体存在感知、动作追踪和生命体征监测，隐私友好的环境监控方案。

## 核心原理：WiFi 也能"看见"你

WiFi 信号在室内传播时会因墙壁、家具、人体反射形成多条传播路径（多径效应）。当人体移动时，这些反射路径的长度会发生变化，导致接收信号相位和幅度产生微妙偏移。通过分析 CSI（Channel State Information，信道状态信息），可以提取出与人体活动相关的特征信号。

```
发射端（路由器 WiFi）→ 人体反射/散射 → 接收端（任意WiFi设备）
                    ↓
           CSI 波形变化分析
                    ↓
           呼吸频率 · 运动轨迹 · 存在检测
```

这与 5G/毫米波雷达的感知原理类似，但完全复用现有 WiFi 基础设施，无需专用传感器。

## 核心功能

### 呼吸频率检测（Respiratory Monitoring）

通过分析 CSI 信号的微多径变化，提取与呼吸节律匹配的周期性信号：

- 非接触式，无需穿戴设备
- 检测精度：典型误差 ±2 BPM
- 适用于：睡眠监测、老人护理、婴儿监护

### 人体存在感知（Presence Detection）

判断空间内是否有人（静止或移动）：

- 穿透墙壁检测（取决于发射功率和频率）
- 区分静态存在和移动
- 响应延迟：< 1 秒

### 动作轨迹追踪（Motion Tracking）

追踪空间内单人或多人移动轨迹：

- 基于多天线（mIMO）信号相位差的空间定位
- 无需人员佩戴任何设备
- 适用于：智能家居自动化、安防异常检测

### 生命体征综合监测

RuView 的终极目标是将 WiFi 感知变成一个"虚拟健康仪表盘"：

- 心率变异性（HRV）估算
- 呼吸间隔异常检测
- 睡眠质量评估（入睡时间、清醒次数、睡眠效率）

## 技术实现

### CSI 数据采集

```python
# 通过 WiFi 网卡驱动获取 CSI 数据
# 支持 Intel IWL5300、Thinkpad AC系列、Qualcomm Atheros 等
from ruview import CSIReceiver

receiver = CSIReceiver(interface='wlan0', channel=36)
csi_data = receiver.receive(duration=30)  # 收集30秒数据
```

### 信号处理流程

```
原始 CSI → 带通滤波（0.1-0.5Hz 呼吸带）→ 去噪（Kalman/小波）→ FFT 提取频率 → 人体活动分类
```

### 实时可视化

RuView 提供实时 Web 仪表盘，支持：
- CSI 波形实时显示
- 呼吸曲线和频率统计
- 空间热力图（人员分布）
- 历史趋势（心率/呼吸/活动量）

## 安装

```bash
# 源码安装
git clone https://github.com/ruvnet/RuView.git
cd RuView
cargo build --release

# 或通过 PyPI（如果已发布）
pip install ruview
```

### 硬件要求

- **WiFi 网卡**：需要支持 Monitor 模式和 CSI 提取（Intel IWL5300/AX200/MT7921 等，需查表）
- **多天线路由器**（建议）：增强空间分辨率
- **Linux**：当前主要支持 Linux，macOS/Windows 通过 Docker 可运行核心算法

## 局限性

- **需要设备**：发射端和接收端需同步在线，信号路径需穿过监测区域
- **环境敏感**：大型金属物体和大面积水体会干扰多径分析
- **精度限制**：无法达到医疗级监测精度，仅作为健康趋势参考
- **安装复杂度**：比智能摄像头配置难度更高

## 总结

RuView 展示了 WiFi 感知技术的实用化潜力——用一台路由器和普通网卡，在完全不侵犯视觉隐私的前提下，实现呼吸监测、人体存在感知和空间追踪。Rust 实现确保了实时性能，模块化设计方便嵌入到智能家居和健康监测系统中。

对于关注隐私的智能家居开发者和健康监测研究者，RuView 是一个值得关注的技术方案。
