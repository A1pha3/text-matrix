# Awesome Free Apps：跨平台免费软件精选列表

## 平台筛选

项目支持按平台过滤，共有以下筛选维度：

- `Windows-only`：仅 Windows 可用
- `macOS-only`：仅 macOS 可用
- `Linux-only`：仅 Linux 可用
- `Open-source-only`：仅开源项目
- `Recommended-only`：编辑推荐

移动端另有 `MOBILE.md` 主文件，支持 Android-only 和 iOS-only 筛选。

## 典型收录应用

### 音频类

- **FXSound**：Windows 音效增强工具，支持音量、低音提升和现代 UI
- **JACK Audio**：音频服务器，支持程序间音频路由和混音，跨平台
- **EarTrumpet**：Windows 高级音量控制工具，支持按应用独立管理音量
- **Ambie**：白噪音和自然音应用，提升专注力，Windows 可用

### 开发工具类

包含 API 开发（如 Postman 的免费替代）、数据库管理（MySQL、PostgreSQL、SQLite 客户端）、网络抓包分析、虚拟机（VirtualBox 等）、游戏引擎等子分类。

### 安全类

- 杀毒软件
- 密码管理器
- 广告与追踪器拦截工具

### 视频类

- 视频编辑器
- 视频播放器
- 录屏与直播工具
- 视频格式转换与压缩工具

## 使用方式

这个项目本身就是一个静态列表，使用方式很直接：

1. 浏览 `README.md` 或对应平台的筛选文件
2. 点击应用名称跳转到官网或 GitHub 仓库
3. 开源项目标注 🟢 时可以查看源码

移动端用户直接查阅 `MOBILE.md`，桌面端用户使用主 README 的平台筛选标签。

## 与同类项目的关系

Awesome Free Apps 属于 Awesome 列表家族的一员。类似的还有 awesome-open-source、awesome-selfhosted 等。项目强调"免费"而非"开源"，因此收录了一些闭源免费软件（如 FXSound）。如果只关注开源替代品，可以使用 `Open-source-only` 筛选标签。
