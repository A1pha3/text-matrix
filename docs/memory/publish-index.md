## 搜索指南

**写文章前必查：**

1. 先用项目名搜索本文档：
   ```
   grep "项目名" publish-index.md
   ```

2. 如果找到，检查 `Commit` 列：
   - 如果只有一个 Commit：文章从未更新
   - 如果有多个 Commit（如 RuView 有 69948e6 和 8513e88）：有更新版本

3. 判断是新建还是更新：
   - 已有内容过旧或需要大改 → 更新现有文章
   - 师父明确要求"重新写" → 新建（保留旧版本）
