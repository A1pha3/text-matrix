# Text Matrix 内容策略

## 定位

Text Matrix 是一个面向中文开发者的 AI Agent、AI 开发工作流与开源工具知识库。

## 核心读者

- 开发者
- 独立开发者
- AI 工具重度使用者
- 对 AI Agent 与工程工作流感兴趣的读者

## 核心栏目

- 技术笔记：长期可检索、可复用的深度内容
- 行业快讯：高频更新的信息入口
- 视频精读：高价值视频内容的整理与提炼

## 辅助栏目

- 财富自由：围绕独立开发、商业化与长期积累的内容
- 思考与随笔：品牌表达与长期观察，不作为主增长栏目

## 发文判断

如果内容的主要价值来自长期可复用，优先进入技术笔记。

如果内容的主要价值来自时效性，优先进入行业快讯。

如果内容本质上是对视频内容的结构化整理，进入视频精读。

如果内容与开发工作流、独立开发或长期积累相关，但不适合作为主栏目长文，可视情况进入辅助栏目。

## 执行原则

- 优先做能沉淀为专题页、系列页和 evergreen 内容的选题
- 快讯服务于覆盖与触达，不挤占首页核心心智
- 深度内容要能被搜索、引用和二次分发
- 新栏目只有在能形成稳定供给和明确读者价值时才扩张
